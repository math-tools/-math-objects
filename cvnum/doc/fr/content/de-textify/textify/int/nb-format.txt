========================
Type de ¨nbres supportés
========================

Par défaut, la langue est l'anglais de Grande-Bretagne. Nous verrons dans la section suivante comment changer de langues.
Ce qui nous intéresse dans cette section, c'est d'indiquer les formats utilisables pour un entier à nommer.
Commençons par deux cas simples via le code suivant qui affichera deux fois de suite le texte "one hundred and twenty-three".

python::
    from cvnum import textify

    intname = IntName()

    # Un nombre de type ``int``.
    print(intname.nameof(123))

    # Un nombre de type ``str``.
    print(intname.nameof("123"))


Plus généralement, on peut utiliser tout objet ¨python dont la représentation via python::``str(...)`` est une écriture valide d'un entier relatif.
Voici un code "farfelue" qui affichera le texte "minus one hundred and twenty-four".

python::
    from cvnum import textify

    class MyOppositeDoubleIntLike:
        def __init__(self, val):
            self.val    = val
            self.double = 2*val

        def __str__(self):
            return f"- {self.double}"

    intname      = IntName()
    myoppodouble = MyOppositeDoubleIntLike(62)
    print(intname.nameof(myoppodouble))
